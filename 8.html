Понимание Операторов Python

Целочисленное Деление и Остаток

Целочисленное Деление

Целочисленное деление выполняется с помощью оператора //. Оно возвращает целую часть результата деления.

Пример:

# Целочисленное деление
result = 7 // 2  # результат: 3
Остаток

Оператор остатка % возвращает остаток от деления.

Пример:

# Остаток
remainder = 7 % 2  # результат: 1
